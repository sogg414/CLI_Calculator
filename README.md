# CLI Calculator (Group X)

A simple terminal calculator written in Go.

## Features
- Basic operations: `+`, `-`, `*`, `/`.
- Extensions: Exponentiation (`^`), square root (`sqrt`).
- Error handling (division by zero, invalid input).
